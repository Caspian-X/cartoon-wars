## Godot
This is a godot project. 

Always use the godot best practices.

## Gamespec

The [gamespec.md](./gamespec.md) is a file that defines states of the game, flow, rules, etc.

Always reference the gamespec to see know how the game works and how parts of it that you may be updating/adding fit into the current state of the game.

After implementing changes you need to check if you should update the gamespec to stay up to date.
